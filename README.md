# Insert markdown color
This plugins makes it able to insert a code snippet for 
coloring text in a markdown file. 

# Usage
1) Select the text to color<br>
2) Press **CTRL + SHIFT + P**. <br>This opens the **Command dialog**
3) Search for the command: ```Markdown Color: Insert markdown color```
4) Type the color you want.<br>
The color has to be valid. Choose a CSS valid color.
## Example 

![](/resources/markdown_color_insert_tutorial.gif)

```html
<span style=color:red>Some red text</span>
<span style=color:orange>Some orange text</span><br>
<span style=color:green>Some green text</span>
```
